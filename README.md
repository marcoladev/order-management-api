# Order Management API

A backend application built with .NET 9 following Clean Architecture principles.

## Features

- Create Orders
- Retrieve Orders
- Cancel Orders
- Entity Framework Core
- MySQL
- FluentValidation
- Unit Testing
- RabbitMQ (coming soon)

## Technologies

- .NET 9
- ASP.NET Core Web API
- Entity Framework Core
- MySQL
- xUnit
- Swagger
